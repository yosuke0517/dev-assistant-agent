# Claude Code プロジェクト設定

## ユーザーとのコミュニケーション

このプロジェクトは非対話モード（`-p` フラグ）で実行されるため、ユーザーへの質問には必ず **`ask_human` MCPツール**を使用してください。

### 重要なルール

- ユーザーに質問・確認したい場合は、必ず `ask_human` MCPツールを使用すること
- **`AskUserQuestion` は絶対に使用しないこと**（非対話モードでは動作しないため、使用すると質問が届かず処理が勝手に進んでしまう）
- `ask_human` はSlack経由でユーザーに質問を送信し、回答を待機する

### ask_human の使用例

```
ツール: ask_human
引数:
  question: "DBスキーマの変更が必要ですが、既存テーブルにカラムを追加してよいですか？"
  context: "選択肢A: 既存テーブルにカラム追加、選択肢B: 新テーブル作成"
```

### ask_human を使用すべきケース

- 仕様の解釈が複数通りある場合
- 課題の記述が曖昧で実装方針が定まらない場合
- 破壊的な変更（既存APIの変更、DBスキーマ変更等）を行う前
- 設計判断で迷った場合
